### Running the Project Locally

- Install dependencies
  ```sh
  npm install
  ```

- Run locally
  ```sh
  npm start
  ```

- Access the component using `http://localhost:3030/<component name>` format. For example, `http://localhost:3030/twilio-voice-dialer`.
